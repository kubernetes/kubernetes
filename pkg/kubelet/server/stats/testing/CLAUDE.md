# Package: testing

## Purpose
The `testing` package provides mock implementations of the stats package interfaces for unit testing.

## Key Types/Structs

- **MockProvider**: Auto-generated mock for the Provider interface using testify/mock. Provides mock implementations for all stats provider methods.
- **MockSummaryProvider**: Auto-generated mock for the SummaryProvider interface.

## Key Functions

- **NewMockProvider**: Creates a new MockProvider instance with testing cleanup.
- **NewMockSummaryProvider**: Creates a new MockSummaryProvider instance with testing cleanup.

## Design Notes

- Generated by mockery (github.com/vektra/mockery) - do not edit manually.
- Uses testify/mock for mock expectations and assertions.
- Each mock method has corresponding Expecter types for fluent expectation setup.
- Mocks support Run, Return, and RunAndReturn patterns for flexible test setup.
