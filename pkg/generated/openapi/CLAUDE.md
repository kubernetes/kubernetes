# Package: openapi

## Purpose
This package contains generated OpenAPI schema definitions for all Kubernetes API types. The schemas are used by the API server to serve OpenAPI documentation and enable API discovery.

## Key Files

- **zz_generated.openapi.go**: Auto-generated file containing GetOpenAPIDefinitions() function
- **doc.go**: Package documentation

## Key Functions

- **GetOpenAPIDefinitions()**: Returns a map of type names to OpenAPI definition specs

## Design Notes

- Generated by openapi-gen from Go type definitions with +k8s:openapi-gen tags
- Schemas include validation rules, descriptions, and type information
- Used by API server to serve /openapi/v2 and /openapi/v3 endpoints
- Supports both OpenAPI v2 and v3 schemas via extension markers
- Do not edit zz_generated.openapi.go manually - regenerate with make update
