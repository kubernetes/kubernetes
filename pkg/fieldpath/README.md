# fieldpath 目录说明

`pkg/fieldpath` 提供按字段路径从 Kubernetes 对象中提取字段值的工具，主要用于支持依赖对象字段引用的功能。

## 关键子目录/源码入口

- `fieldpath.go`：实现字段路径解析和对象字段提取逻辑。
- `fieldpath_test.go`：覆盖支持的字段路径、错误路径和对象类型场景。

## 与 cmd/staging/test 的关系

- kubelet、控制器或 API 处理逻辑可通过本包解析对象字段引用。
- 本目录依赖 Kubernetes API 对象定义，但不是 staging 中的通用字段路径库。
- 包内单元测试是主要覆盖来源，相关功能也可能在 pod 或 downward API 测试中被间接覆盖。

## 开发/测试注意事项

- 修改字段路径语义时，要注意已有用户可见行为和错误信息兼容性。
- 新增支持字段应明确对象类型、版本和空值处理规则。
- 建议运行本包测试，并补充调用方场景测试。
